# Netflix-Movies-Shows-Analysis

Data analytics project exploring Netflix movies and TV shows dataset using Python, SQL, and Power BI. Focused on data cleaning, analysis, and visualization to understand content trends.

## Analysis Done:

* Data cleaning and preprocessing
* Movies vs TV Shows comparison
* Genre analysis
* Year-wise content trend
* Country-wise content distribution

## Key Findings:

* Movies are more than TV shows on Netflix

* Content increased significantly after 2015

* Certain genres dominate the platform

* Content is concentrated in a few countries
